**OWASP BLT (Bug Logging Tool)** is a comprehensive bug bounty and vulnerability management platform. This organization contains all official OWASP BLT repositories.

| Repository | Description | Language |
|------------|-------------|----------|

| **[BLT-Flutter](https://github.com/OWASP-BLT/BLT-Flutter)** | Official mobile app for bug hunting on the go | Dart |
| **[BLT-Extension](https://github.com/OWASP-BLT/BLT-Extension)** | Chrome extension for screenshot reporting | JavaScript |
| **[BLT-Action](https://github.com/OWASP-BLT/BLT-Action)** | GitHub Actions for automatic issue assignment | JavaScript |
| **[OWASP-BLT-API](https://github.com/OWASP-BLT/OWASP-BLT-API)** | Cloudflare workers API that interfaces with the PostgreSQL database | TypeScript |
